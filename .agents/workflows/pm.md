# PM Agent 실행 워크플로우

1. `.claude/agents/pm-agent.md` 파일을 읽어서 PM 에이전트의 역할과 지침을 파악합니다.
2. `docs/prd.md`와 `CLAUDE.md`, `AGENTS.md`를 읽어 프로젝트 전체 컨텍스트와 목표를 파악합니다.
3. 사용자가 지시한 목표(예: "로비 화면 만들어줘", "Task 2 진행해줘")를 분석하고, 현재 프로젝트 상태(기존 파일, 에러 현황 등)를 확인합니다.
4. 사용자에게 앞으로 진행할 **실행 계획을 먼저 보고**합니다.
5. 사용자가 동의하면, 의존성 순서에 따라 적절한 에이전트(예: ui-agent, scene-agent 등)의 역할을 대신 수행하며 지시된 Task를 진행합니다.
6. 작업 완료 후, 변경된 파일 목록과 현재 상태(정상/에러), 그리고 다음 단계를 보고합니다.
